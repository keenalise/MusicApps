## Step 10: Troubleshooting Common Issues

### Issue: "SessionsScreen is not found"
Solution: Ensure SessionsScreen.kt is in ui/screens/ and properly imported

### Issue: "viewModel() returns null"
Solution: Make sure viewmodel() is imported from:
```kotlin
import androidx.lifecycle.viewmodel.compose.viewModel
```

### Issue: Bottom nav bar doesn't show Sessions tab
Solution: Check:
1. Screen.Sessions added to Screen sealed class
2. bottomNavItems updated
3. BottomNavigationItem added for Sessions
4. Navigation route matches exactly

### Issue: "Cannot access sessionRepository"
Solution: Use public methods only:
```kotlin
// DON'T do this (repository is private):
sessionViewModel.sessionRepository.createSession(...)

// DO this instead:
sessionViewModel.createNewSession("name")
sessionViewModel.switchToSession(id)
```

## Common Code Patterns

### Pattern 1: Switch to Session and Play
```kotlin
Button(onClick = {
    sessionViewModel.switchToSession(sessionId, autoPlay = true)
}) {
    Text("Play This Session")
}
```

### Pattern 2: Create and Immediately Switch
```kotlin
LaunchedEffect(Unit) {
    sessionViewModel.createNewSession("New Playlist")
    // Slight delay for DB insert
    delay(100)
    // Session ID needed here - requires more logic
    // Best approach: use callback
}
```

### Pattern 3: Observe Sessions and Update UI
```kotlin
val allSessions by sessionViewModel.allSessions.collectAsStateWithLifecycle()
val activeSession by sessionViewModel.activeSession.collectAsStateWithLifecycle()

LazyColumn {
    items(allSessions) { session ->
        SessionItem(
            session = session,
            isActive = session.id == activeSession?.id
        )
    }
}
```

### Pattern 4: Handle Errors
```kotlin
val errorMessage by sessionViewModel.errorMessage.collectAsStateWithLifecycle()

if (errorMessage != null) {
    LaunchedEffect(errorMessage) {
        // Show toast or snackbar
        Toast.makeText(context, errorMessage, Toast.LENGTH_SHORT).show()
        // Clear after showing
        viewModel.clearError()
    }
}
```
