GETTING THE VIEWMODEL
═════════════════════

In Composable:
  val viewModel: SessionManagementViewModel = viewModel()

In Fragment/Activity:
  val viewModel: SessionManagementViewModel by viewModels()

═════════════════════════════════════════════════════════════════

CREATING SESSIONS
═════════════════

Create empty session:
  viewModel.createNewSession("Playlist Name")

Create session with songs:
  // Use SessionQueueManager for this
  val queueManager = SessionQueueManager(repository)
  val sessionId = queueManager.createSessionWithSongs(
    "My Favorites",
    listOf(song1, song2, song3)
  )

═════════════════════════════════════════════════════════════════

SWITCHING SESSIONS
══════════════════

Switch to session (pause if playing):
  viewModel.switchToSession(sessionId = 123)

Switch and auto-play:
  viewModel.switchToSession(sessionId = 123, autoPlay = true)

═════════════════════════════════════════════════════════════════

MANAGING SESSIONS
═════════════════

Rename session:
  viewModel.renameSession(sessionId = 123, newTitle = "New Name")

Delete session:
  viewModel.deleteSession(sessionId = 123)

═════════════════════════════════════════════════════════════════

OBSERVING STATE
═══════════════

Get all sessions (reactive):
  val allSessions by viewModel.allSessions.collectAsStateWithLifecycle()

Get active session:
  val activeSession by viewModel.activeSession.collectAsStateWithLifecycle()

Get error messages:
  val error by viewModel.errorMessage.collectAsStateWithLifecycle()

Get creating status:
  val isCreating by viewModel.isCreatingSession.collectAsStateWithLifecycle()

═════════════════════════════════════════════════════════════════

ADVANCED - QUEUE MANAGEMENT
═══════════════════════════

Init queue manager:
  val queueManager = SessionQueueManager(sessionRepository)

Add songs to session:
  queueManager.addSongsToSession(sessionId, listOf(song1, song2))

Replace entire queue:
  queueManager.replaceSessionQueue(sessionId, newSongsList)

Get playable URI:
  val uri = queueManager.getPlayableUri(sessionId, queueIndex)

═════════════════════════════════════════════════════════════════

SESSION STATE OBJECT
════════════════════

PlaybackSessionEntity has:
  id: Long                          // Unique ID
  title: String?                    // Session name
  createdAt: Long                   // Creation timestamp
  updatedAt: Long                   // Last update timestamp
  currentIndex: Int                 // Current track (0-based)
  lastPositionMs: Long              // Playback position (milliseconds)
  playbackState: Int                // 0=STOPPED, 1=PAUSED, 2=PLAYING
  repeatMode: Int                   // Repeat setting
  shuffleModeEnabled: Boolean       // Shuffle toggle
  isActive: Boolean                 // Currently active session

═════════════════════════════════════════════════════════════════

QUEUE ITEM STATE OBJECT
═══════════════════════

QueueItemEntity has:
  id: Long                    // Unique ID
  sessionId: Long             // Parent session ID
  mediaId: String             // Song identifier
  uri: String                 // File URI or content:// path
  title: String?              // Song title
  artist: String?             // Artist name
  album: String?              // Album name
  durationMs: Long?           // Song duration in milliseconds
  positionInQueue: Int        // Position in queue (0-based)
  extrasJson: String?         // Reserved for future use

COMMON PATTERNS
═══════════════

Pattern 1: Display all sessions with active highlight
───────────────────────────────────────────────────
val allSessions by vm.allSessions.collectAsStateWithLifecycle()
val activeSession by vm.activeSession.collectAsStateWithLifecycle()

LazyColumn {
  items(allSessions) { session ->
    SessionCard(
      session = session,
      isActive = session.id == activeSession?.id,
      onClick = { vm.switchToSession(session.id, autoPlay = true) }
    )
  }
}

Pattern 2: Handle creation with error display
─────────────────────────────────────────────
var sessionName by remember { mutableStateOf("") }
val isCreating by vm.isCreatingSession.collectAsStateWithLifecycle()
val error by vm.errorMessage.collectAsStateWithLifecycle()

Button(
  onClick = { vm.createNewSession(sessionName) },
  enabled = sessionName.isNotBlank() && !isCreating
) {
  Text("Create")
}

if (error != null) {
  Text(error!!, color = MaterialTheme.colorScheme.error)
}

Pattern 3: Auto-switch to first session on load
────────────────────────────────────────────────
val allSessions by vm.allSessions.collectAsStateWithLifecycle()

LaunchedEffect(allSessions) {
  if (allSessions.isNotEmpty() && activeSession == null) {
    vm.switchToSession(allSessions.first().id, autoPlay = false)
  }
}

ERROR MESSAGES
══════════════

vm.errorMessage provides:
  - "Session not found: $sessionId"
  - "Failed to create session: ${e.message}"
  - "Failed to delete session: ${e.message}"
  - "Failed to rename session: ${e.message}"
  - "Failed to update queue: ${e.message}"
  - "Failed to switch session: ${e.message}"

Clear errors:
  vm.clearError()

LIFECYCLE INTEGRATION
═════════════════════

In ViewModel.onCleared():
  - Automatically saves active session state
  - Releases ExoPlayer
  - Cancels all coroutines

In Fragment/Activity onDestroy():
  - ViewModel is automatically cleared
  - Session state persisted

DATABASE QUERIES (Reference)

COMMON ERRORS & FIXES
═════════════════════

Error: "Session not found"
Fix: Verify sessionId exists before switching

Error: "Cannot access sessionRepository"
Fix: Use viewModel public methods (createNewSession, switchToSession)
     Don't access private repository directly

Error: "ExoPlayer not initialized"
Fix: Ensure SessionManager is created with valid ExoPlayer instance
     Check that player is not released

Error: "Position not restoring"
Fix: Verify saveActiveSessionState() called before switching
     Check currentIndex in DB is correct
     Ensure player.prepare() called after setMediaItems()
