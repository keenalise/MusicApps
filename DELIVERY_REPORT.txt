API OVERVIEW
════════════════════════════════════════════════════════════════

Getting Started:

Create Session:
  vm.createNewSession("My Playlist")

Switch Session:
  vm.switchToSession(123, autoPlay = true)

Manage:
  vm.renameSession(123, "New Name")
  vm.deleteSession(123)

Observe:
  val sessions = vm.allSessions.collectAsStateWithLifecycle()
